# Excel Double Entry System

Ein einfaches, verständliches und für Schulungszwecke entwickeltes System zur doppelten Buchführung auf Basis von **Excel + VBA**.

Dieses Projekt enthält:
- Automatisches Erfassen von Geschäftsfällen
- Verbuchen der Beträge in zugehörige T-Konten
- Automatische Vergabe von Buchungsnummern
- Grundlagen von Bilanz und Gewinn- und Verlustrechnung
- Dynamische Tabellenstrukturen für Soll/Haben-Buchungen

Das Projekt dient ausschließlich zu **Lern- und Demonstrationszwecken**, um die Abläufe der doppelten Buchhaltung praxisnah nachvollziehen zu können.
